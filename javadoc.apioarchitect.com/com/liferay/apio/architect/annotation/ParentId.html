Annotation Type ParentId


  • @Retention(value=RUNTIME)
 @Target(value=PARAMETER)
public @interface ParentId
    Defines an annotation that provides information about a parent type ID.

    This annotation should always be used on an ActionRouter method parameter representing a parent resource ID.

    A PathIdentifierMapper for the ID type must exist for Apio to automatically convert it from the request.
