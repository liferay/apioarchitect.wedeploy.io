public interface ReusableNestedCollectionRouter<T,S,U extends Identifier<S>,W>
    Represents the mapping between HTTP requests for a reusable nested collection resource and the functions that perform those requests.

    A reusable nested collection resource is the type of resource that isn't bound to a parent resource.

    The type param provided for the collection ID must be unique in the whole application.

    You can add the different supported routes for the nested collection resource via the collectionRoutes(NestedCollectionRoutes.Builder) method.

    The union of an instance of this interface with a Representable creates a complete resource that behaves as its own API.
