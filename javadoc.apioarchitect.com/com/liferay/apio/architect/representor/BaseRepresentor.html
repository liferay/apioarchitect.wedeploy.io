Interface BaseRepresentor<T>

  • Type Parameters:
    T - the model's type
    All Known Subinterfaces:
    NestedRepresentor<T>, Representor<T>

    @ProviderType
public interface BaseRepresentor<T>
    Base class for Representors.

    Descendants of this class holds information about the metadata supported for a resource.

    Only two descendants are allowed: Representor and NestedRepresentor.
