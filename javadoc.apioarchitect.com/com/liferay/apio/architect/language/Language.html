      • getLocales

        default java.util.stream.Stream<java.util.Locale> getLocales()
        Deprecated. 
        Returns the Stream of the request's preferred Locale, in decreasing order.

        The stream values start with the first locale added on the Accept-Language header and continue with the rest of the header.

        If the request doesn't have an Accept-Language header, this method returns an Stream containing the default locale for the server.

        Returns:
        the Stream of the request's preferred Locale, if the Accept-Language header is present; otherwise returns the Stream containing the server's default locale
